# campaign-manager

A campaign manager app built using Metal.js.

## Setup

1. Install NodeJS >= [v5.0.0](http://nodejs.org/dist/v5.0.0/), if you don't have it yet.

2. Install local dependencies:

  ```
  npm install
  ```

3. Run the server:

  ```
  npm run start
  ```

4. Open the site using the url the previous command indicated (something like **localhost:8080**).
